# JaffDb

`jaff.drivers.sqlite.JaffDb`

Subclass of `Db` pre-configured to connect to the internal JAFF database bundled with the package (`jaff/db/jaff.db`). No path argument is needed.

## Constructor

`#!python JaffDb()`

Automatically resolves the path to the bundled JAFF database.
