# table_from_csv

`#!python table_from_csv(name, file, delimiter)`

Creates a table from a CSV file by reading it into a DataFrame first.

**Parameters**

**name** : _str_
: Table name.

**file** : _Path_
: CSV file path.

**delimiter** : _str_
: Column delimiter.

**Returns**

_Table_
: Table wrapper for the newly created table.
