# has_any_species

`#!python has_any_species(species)`

Returns `True` if any of the given species appear in either reactants or products.

**Parameters**

**species** : _Specie, str, or list[Specie or str]_
: Species to check for.

**Returns**

_bool_
: `True` if at least one of the given species appears in this reaction.
