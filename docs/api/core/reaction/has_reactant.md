# has_reactant

`#!python has_reactant(species)`

Returns `True` if all given species are present in the reactants.

**Parameters**

**species** : _Specie, str, or list[Specie or str]_
: Species to check for.

**Returns**

_bool_
: `True` if all given species are among the reactants.
