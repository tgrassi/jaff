# compare_species

`#!python compare_species(other, verbosity=1)`

Compares species between this network and another, logging common and unique species.

**Parameters**

**other** : _Network_
: Network to compare against.

**verbosity** : _int, optional_
: `1` prints lists of differing species. Default `1`.
