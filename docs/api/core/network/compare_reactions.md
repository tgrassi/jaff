# compare_reactions

`#!python compare_reactions(other, verbosity=1)`

Compares reactions between this network and another, logging common and unique reactions.

**Parameters**

**other** : _Network_
: Network to compare against.

**verbosity** : _int, optional_
: `1` prints lists of differing reactions. Default `1`.
