# photo_reaction_indices

`#!python photo_reaction_indices()`

Returns the zero-based catalogue indices of all photo-reactions, in ascending index order. Use these indices to slice other bulk-accessor vectors so that only the photo-reaction entries are retained.

**Returns**

_Vector[int]_
: Zero-based catalogue index for each photo-reaction, in ascending order.
