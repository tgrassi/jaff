# exploded

`#!python exploded(ne=False)`

Returns the exploded element lists for all species in the collection.

**Parameters**

**ne** : _bool, optional_
: If `True`, excludes `"e-"`. Default `False`.

**Returns**

_Vector[list[str]]_
: Exploded element lists for each species.
