# elements

`#!python elements(ne=False)`

Returns [`Elements`](../elements/index.md) objects for all species in the collection.

**Parameters**

**ne** : _bool, optional_
: If `True`, excludes `"e-"`. Default `False`.

**Returns**

_Vector[Elements]_
: Elements objects for each species.
